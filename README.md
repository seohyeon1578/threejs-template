# Three.js-Template
Three.js 라이브러리를 빠르게 사용하기 위한 기본 템플릿 입니다.

## Getting Start

Install dependencies 

```
yarn
```

start a local server:

```
yarn dev
```

Local Server will open on:

```
http://localhost:8080/
```

Create the build:

```
yarn build
```

## Features

- Three.JS
- webpack
- babel
